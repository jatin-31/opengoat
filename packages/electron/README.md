# @opengoat/desktop

Electron desktop app for OpenGoat.

## Stack

- Electron Forge + Vite
- React + Tailwind
- shadcn-style component primitives
- Zustand state management
- tRPC over IPC (`electron-trpc`)

## Run

From repo root:

```bash
npm install
npm run desktop:dev
```

## Package

```bash
npm run desktop:package
npm run desktop:make
```
