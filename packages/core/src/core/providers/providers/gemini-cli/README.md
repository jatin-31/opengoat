# Gemini CLI Provider

- Type: CLI
- Command: `gemini`
- Env override: `GEMINI_CMD`
- Invocation mode: headless prompt (`gemini -p ... --output-format json --yolo`)
- Session resume: supported (`gemini --resume <session-id>`)
- Auth command: not supported (configure via env/settings)
- External agent creation: not supported
- External agent deletion: not supported
