# Cursor Provider

- Type: CLI
- Command: `cursor` (or `cursor-agent` via `CURSOR_CMD`)
- Env override: `CURSOR_CMD`
- Auth command: `cursor agent login`
- Invocation mode: headless print (`cursor agent --print --output-format json --force --sandbox disabled --trust`)
- Session resume: supported (`cursor agent --resume <session-id>`)
- External agent creation: not supported
- External agent deletion: not supported
